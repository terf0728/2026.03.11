# 로또 번호 추천

토스 스타일의 로또 번호 추천 웹 서비스입니다.

## 기능

- 1~45 중 6개 본번호 + 1개 보너스 번호 추천
- 사탕머신 형태의 원형 돔 추첨기 애니메이션
- 1/3/5/10세트 선택 가능
- 반응형 디자인

## 실행 방법

`index.html`을 브라우저에서 열거나, 로컬 서버로 실행합니다.

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

## 기술 스택

- HTML, CSS, JavaScript
- 토스 디자인 톤 적용
